Explication

FitnessCard est une application permettant de personnaliser un circuit trainingsur la base de 4 exercices (4 couleurs de carte)

Le principe est simple :

  • Choisisez un exercice pour chaque couleur
  • Appuyer sur commencer
  • Les cartes vont sortir aléatoirement avec une valeur (exemple: 3 de coeur a une valeur de 3
  • Réaliser le nombre de répétition lié à cette valeur ou effectué la durée de l'exercice si c'est un exercice de type temps
    • elui ci est accecible dans les paramètre de l'application
  • Lorsque c'est un exercice de type temps, le temps correspond à la valeur de la carte multiplié par le multiplicateur de temps (par défaut 6). C
