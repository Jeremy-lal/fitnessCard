Explication

FitnessCard est une application permettant de personnaliser un circuit training sur la base de 4 exercices (4 couleurs dans un paquet de cartes)

Le principe est simple :

  • Choisisez un niveau de difficulté
  • Choisisez un exercice pour chaque couleur
  • Appuyez sur commencer
  • Les cartes vont sortir aléatoirement avec une valeur (exemple: 3 de coeur a une valeur de 3)
  • Réaliser le temps ou le nombre de répétition indiqué.
  • Lorsque c'est un exercice de type temps, le temps correspond à la valeur de la carte multiplié par le multiplicateur de temps.
  • Lorsque c'est un exercice de type répétition, le nombre de répétition correspond à la valeur de la carte multiplié par le multiplicateur de répétitiion.
  • Lorsque vous avez fini l'exercise, faire défiler vers la gauche pour avoir le suivant
  • Quand vous avez fini le paquet de cartes ou que vous voulez arrêter, cliquez sur Finir

Valeur des multiplicateurs en fonction du niveau :

  • Débutant = répétition : x1, temps : x2
  • Facile = répétition : x1, temps : x4
  • Moyen = répétition : x1, temps : x6
  • Difficile = répétition : x2, temps : x8
  • Expert = répétition : x2, temps : x10

FitnessCard is an application to customize a training session based on 4 exercises (4 suits in a card deck)

The principle is simple :

  • Choose your level
  • Choose un exercice for each suits
  • Click on Start
  • The cards will come out randomly with a value (example: 3 of heart and a value of 3)
  • Make the time or number of repetitions indicated.
  • When it is a time type exercise, the time corresponds to the value of the card multiplied by the time multiplier.
  • When it is a repetition exercise, the repetition number is the value of the card multiplied by the repetition multiplier.
  • When you have finished the exercise, scroll to the left to have the following
  • When you have finished the card deck or want to stop, click on Finish

Value of the multipliers according to the level :

  • Beginnner = repetition : x1, time : x2
  • Easy = repetition : x1, time : x4
  • Middle = repetition : x1, time : x6
  • Hard = repetition : x2, time : x8
  • Expert = repetition : x2, time : x10
